La necessit√† di caricare sul server ftp gli ultimi commits mi ha portato a scoprire git-ftp. Si installa su Mac,Win e Linux. Per installare il pacchetto su debian, consiglio di aggiungere il ppa ubuntu dello sviluppatore ed importare la chiave in modo da avere sempre il pacchetto aggiornato. Per configurare le credenziali di accesso bisogna: {% highlight bash %} $ git config git-ftp.user john $ git config git-ftp.url ftp.example.com/myfolder $ git config git-ftp.password secr3t {% endhighlight %} poi per caricare il tutto {% highlight bash %} $ git ftp push {% endhighlight %} Se non riuscite a connettervi, verificate che il firewall del vostro hosting non stia bloccando cURL. In ogni caso, gli sviluppatori stanno lavorando ad una nuova versione che utilizza lftp invece che cURL. Dopo aver fatto questo, veniamo agli alias per git. Questi sono quelli che ho aggiunto io al mio file .bashrc {% highlight bash %} alias ga='git add' alias gp='git push' alias gfp='git ftp push' alias gl='git log' alias gs='git status' alias gd='git diff' alias gdc='git diff --cached' alias gm='git commit -m' alias gma='git commit -am' alias gb='git branch' alias gc='git checkout' alias gra='git remote add' alias grr='git remote rm' alias gpu='git pull' alias gcl='git clone' alias glast='git show --pretty="format:" --name-only' {% endhighlight %} L'ultimo alias elenca i file modificati nell'ultimo commit effettuato.
