Mp3 file types Supporting tagline

Ho dovuto realizzare un uploader utlizzando questo plugin nel quale si possono specificare quali tipi di file e' possibile caricare. L'applicazione si concentra sui file audio, da qui la necessita' di elencare i filetypes consentiti.
