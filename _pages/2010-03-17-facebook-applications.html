--- layout: page title: Facebook applications published: false meta: _edit_last: "1" _wp_page_template: single-page-template.php tags: [] type: page status: draft --- Facebook ha un bacino di utenti enorme, difficile da ignorare. Inserire la propria pubblicità in facebook significa sfruttare un network consolidato per la diffusione virale del vostro messaggio. Come si fa? Progettando applicazioni che permettono di interfacciarsi con questa popolare piattaforma. Ci occupiamo della realizzazione di facebook applications scritte tramite le api pubbliche di facebook in php o ruby. Dall'ideazione, alla scelta delle possibili interazioni con l'utente;  dal più semplice pulsante alla più complessa animazione in Flash. Nel caso foste interessati ad un preventivo per una facebook application, contattateci.
